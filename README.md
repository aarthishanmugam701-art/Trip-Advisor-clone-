 #  TripAdvisor Clone

A responsive front-end clone of the TripAdvisor website built using **HTML and CSS**.  
This project replicates the layout, structure, and design of a travel review platform with a clean and modern user interface.

---

## Project Overview

The TripAdvisor Clone is a static travel website designed to showcase destinations, hotels, restaurants, and attractions.  
This project focuses on front-end development concepts such as structured HTML layout, CSS styling, and responsive web design without using JavaScript.

---

##  Features

- Attractive Homepage Design  
- Navigation Bar with Multiple Menu Links  
- Search Bar Section (UI Design)  
- Experience Section 
- Attractions & Things to Do  
- Review-style Card Layout  
- Fully Responsive Design  
- Clean and Organized Code Structure  

---

##  Technologies Used

- HTML5  
- CSS3  
- Flexbox  
- Media Queries
